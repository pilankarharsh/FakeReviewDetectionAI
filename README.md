# Fake Review Detector

A BERT-powered web application that analyzes product reviews and determines whether they are likely fake or genuine. Built with FastAPI and trained on 650k+ reviews with 98.6% validation accuracy.

## Features

- Real-time review classification with confidence scores
- Detailed signal analysis (word count, lexical diversity, exclamation usage, etc.)
- Red flag detection for suspicious patterns
- Interactive web interface
- REST API for integration
- GPU support with automatic CPU fallback

## Quick Start

### Prerequisites
- Python 3.8+
- pip

### Installation

1. **Clone and navigate to project:**
```bash
git clone https://github.com/your-username/fake-review-detector.git
cd fake-review-detector
```

2. **Create virtual environment (recommended):**
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python3 -m venv venv
source venv/bin/activate
```

3. **Install dependencies:**
```bash
pip install -r requirements.txt
```

**Windows users:** If you encounter build errors, use:
```bash
pip install --only-binary=all -r requirements.txt
```

### Running the Application

```bash
cd app
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

Then open **http://localhost:8000** in your browser.

## Usage

### Web Interface
1. Navigate to http://localhost:8000
2. Enter a review text
3. Click "Analyze Review"
4. View the detailed analysis results

### API Usage

**Endpoint:** `POST /predict`

```bash
curl -X POST "http://localhost:8000/predict" \
     -H "Content-Type: application/json" \
     -d '{"text": "This product is amazing! I love it so much!"}'
```

**Response:**
```json
{
  "verdict": "GENUINE",
  "confidence": 93.4,
  "fake_probability": 6.6,
  "genuine_probability": 93.4,
  "signals": {
    "word_count": 8,
    "lexical_diversity": 87.5,
    "exclamation_ratio": 12.5,
    "avg_word_length": 4.0,
    "caps_ratio": 0.0,
    "repetition_score": 0.0
  },
  "red_flags": ["Excessive exclamation marks"]
}
```

## Troubleshooting

**Common Issues:**

1. **"Site can't be reached"** - Use `http://localhost:8000` instead of `http://0.0.0.0:8000`

2. **ModuleNotFoundError** - Install dependencies: `pip install -r requirements.txt`

3. **Build errors on Windows** - Use pre-compiled binaries: `pip install --only-binary=all -r requirements.txt`

4. **Model loading failed** - Ensure all model files exist in `models/bert/`

5. **Port already in use** - Use different port: `python -m uvicorn main:app --port 8001`
